# Gold Ethiopia landing

Marketing / Linktree-style landing page for the **Gold Ethiopia** app (Google Play download + Telegram + Facebook) — designed to be shared as **one link** in ads and bios.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deploy notes

- **Set** `NEXT_PUBLIC_SITE_URL` to your deployed domain so social previews (OpenGraph/Twitter) resolve correctly.
